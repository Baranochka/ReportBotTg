# Telegram-Bot
#### Описание:
Разработана система из двух ___Telegram-ботов___ на ___Python___, предназначенную для централизованного сбора информации от сотрудников и автоматического формирования ежемесячных отчётов для руководства и администраторов.

#### Функциональность и реализованные решения:

- Отдельный бот для сотрудников: пошаговый сбор и валидация данных через диалоговый интерфейс ___Telegram___.

- Административный бот: получение сводных отчётов, контроль заполнения.

- Хранение данных в ___SQLite___ (структурированная схема, CRUD-операции).

- Автоматическое формирование месячных отчётов на основе собранных данных.

- Разграничение ролей и прав доступа (сотрудники / администраторы).

- Обработка ошибок и защита от некорректного ввода.

#### Технологии и инструменты:

- ___Python 3.13.2___

- ___Telegram Bot API___ (___telebot___)

- ___SQLite___

- Cинхронная обработка запросов

Проект реализован полностью самостоятельно: проектирование архитектуры, разработка логики ботов, работа с базой данных и сопровождение.
